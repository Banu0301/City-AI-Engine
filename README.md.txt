# City AI Engine

City AI Engine is a GenAI-powered tool designed to revolutionize urban planning by providing data-driven insights through AI models and interactive visualization tools.

## Features
- City Profiling**: Use city data to generate key insights.
- Footpath Quality Assessment**: Assess footpath conditions using AI and computer vision.

## Technologies
- Flask
- Pandas
- OpenCV
- GenAI models

## How to Run
1. Install the requirements:
    ```
    pip install -r requirements.txt
    ```

2. Run the Flask app:
    ```
    python app.py
    ```

3. Access the app at `http://127.0.0.1:5000/`.
